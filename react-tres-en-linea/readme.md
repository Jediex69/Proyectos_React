# 🎮 Tres en Línea (React)

Juego clásico de **Tres en Línea (Tic-Tac-Toe)** desarrollado con **React** como proyecto de aprendizaje para practicar los fundamentos del desarrollo de interfaces modernas.

## 📋 Descripción

La aplicación permite que dos jugadores compitan alternando turnos hasta conseguir tres fichas consecutivas en horizontal, vertical o diagonal. También detecta automáticamente los empates cuando no quedan movimientos disponibles.

Este proyecto fue desarrollado con un enfoque didáctico para consolidar conceptos fundamentales de React.

## 🚀 Tecnologías utilizadas

* React
* JavaScript (ES6+)
* HTML5
* CSS3

## 🎯 Objetivos de aprendizaje

Durante el desarrollo de este proyecto se trabajaron conceptos como:

* Creación de componentes reutilizables.
* Gestión del estado mediante `useState`.
* Comunicación entre componentes.
* Renderizado condicional.
* Manejo de eventos.
* Actualización dinámica de la interfaz.
* Organización de un proyecto React.

## ▶️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/Jediex69/Proyectos_React.git
```

Accede al proyecto:

```bash
cd Proyectos_React/react-tres-en-linea
```

Instala las dependencias:

```bash
npm install
```

Inicia la aplicación:

```bash
npm run dev
```
